# 🔐 Tehdit Modelleme: Pasaport Kontrolü (Authentication)
Pasaport verilerine erişim JWT (JSON Web Token) ile korunmaktadır. Bir saldırganın 'Broken Object Level Authorization' (BOLA) denemelerine karşı Appwrite'ın sunucu taraflı yetki kontrolü (ACL) analiz edilmiştir.
